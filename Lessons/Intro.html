What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating web pages.

<!DOCTYPE html>
<html>
<head>
</head>
<body>
</body>
</html>

Basic HTML Structure

Every HTML document has a basic structure that includes elements like <html>, <head>, and <body>.

<html>: The root element of an HTML page.
<head>: Contains meta-information about the HTML document.
<body>: Contains the content of the HTML document.

Common HTML Tags

Here are some common HTML tags:

<h1> to <h6>: Header tags
<p>: Paragraph tag
<a>: Anchor tag for links
<img>: Image tag
